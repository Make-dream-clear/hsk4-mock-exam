Conjunctions / 连词

只要 vs 只有

as long as (sufficient) vs only if (necessary) — opposite logic

Comparison / 对比

只要 zhǐyào只有 zhǐyǒu
LogicSufficient condition: A is enough for BNecessary condition: B requires A
Pair只要…就…只有…才…
MeaningAs long as A, then B will happenOnly if A, can B happen
FlexibilityA is one of many ways to get BA is the ONLY way to get B

Examples / 例句

只要: 只要努力学习，就一定能通过考试。
Zhǐyào nǔlì xuéxí, jiù yídìng néng tōngguò kǎoshì.
As long as you study hard, you'll definitely pass. (sufficient)
只有: 只有努力学习，才能通过考试。
Zhǐyǒu nǔlì xuéxí, cái néng tōngguò kǎoshì.
Only by studying hard can you pass. (necessary, no other way)
Quick rule: 只要…就 = A is ENOUGH (optimistic). 只有…才 = A is the ONLY way (stricter). Same topic, different emphasis. This is one of the most frequently tested pairs in HSK 4.

Quick Quiz / 小测验

你有信心，___没有做不到的事。
Confidence is sufficient (optimistic) → 只要…就.
亲自去一趟，___能了解真实情况。
Going in person is the ONLY way → 只有…才.
坚持锻炼，身体___会越来越好。
Exercising is enough to improve → 只要…就.
