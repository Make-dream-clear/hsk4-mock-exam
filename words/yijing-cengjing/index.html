Adverbs / 副词

已经 vs 曾经

already (current state) vs once (past experience no longer true)

Comparison / 对比

已经 yǐjīng曾经 céngjīng
FocusResult still relevant nowPast experience, may not be true now
ImplicationThe state continuesThe state has changed
With 了Often: 已经…了Rarely used with 了
Englishalready, by nowonce, formerly, used to

Examples / 例句

已经: 我已经吃过饭了，不饿。
Wǒ yǐjīng chī guò fàn le, bù è.
I've already eaten — I'm not hungry. (result still relevant)
曾经: 他曾经是一名老师，后来改行做了生意。
Tā céngjīng shì yì míng lǎoshī, hòulái gǎiháng zuò le shēngyi.
He was once a teacher, then switched to business. (no longer true)
Quick rule: 已经 = the result matters NOW (已经毕业了 = graduated and that's my current status). 曾经 = it was true BEFORE but may not be now (曾经住在北京 = used to live in Beijing, but doesn't anymore).

Quick Quiz / 小测验

她在法国留过学，所以法语说得很好。
Past experience (studied in France before) → 曾经.
飞机起飞了，你赶不上了。
Current state (plane has taken off now) → 已经.
这里是一片森林，现在变成了城市。
Was a forest before, no longer true → 曾经.
